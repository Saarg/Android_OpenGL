Arcade Shooter, scroll down.

- Player :
	1 - Render le Player, Triangle pointé vers le haut;
	2 - Deplacer le Player avec l'acceleromètre Droite/Gauche;      <--Require P.1
	3 - Tirer les missiles depuis le Player, à interval régulier ou sur action du joueur; <--Require M.1 P.1;

- Missiles :
	1 - Render les missiles, Triangles piti;
	2 - Deplacer les missiles vers le haut; 				             <-- Require M.1;

- Obstacles :
	1 - Render des Obstacles --> Carré/Rectangles;
	2 - Deplacer les Obstacles vers le bas;					                    <-- Require O.1;
	3 - Collisions avec les Missiles, Point Haut du missile ONLY;		   <--Require M.1 O.1;
	4 - Collisions avec le Player;						                         <-- Require P.1 O.1;

- Game :
	1 - Gérer Victoire Défaite -> mort du Player;                       <-- Require P.1 O.1.4
	2 - Gérer les vagues d'Obstacles ; 					                         <-- Require O.1.2
